# PhishGuard

PhishGuard is a Hybrid Phishing Detection System that combines:

• Machine Learning (Random Forest URL classification)
• NLP based keyword analysis
• Webpage inspection
• Hybrid risk scoring engine

The system analyzes URLs and determines the likelihood of phishing attacks.

## Features

- ML based phishing detection
- URL lexical feature analysis
- NLP keyword detection
- Webpage structure inspection
- Hybrid risk engine
- Interactive dashboard

## Technologies Used

Python  
Flask  
Scikit-learn  
BeautifulSoup  
HTML / CSS  

## Running Locally

Install dependencies:

pip install -r requirements.txt

Run the server:

python app.py

Open browser:

http://127.0.0.1:5000
