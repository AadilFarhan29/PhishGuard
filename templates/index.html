Machine Learning: URL structure classification using the trained Random Forest model.

NLP Analysis: Suspicious terms, brand impersonation, and phishing-related language patterns.

Webpage Inspection: Forms, password fields, iframes, scripts, and external links.

Hybrid Risk Engine: Combines all layers into one final verdict with reasoning.
